##encryption

* DES(Data Encryption Standard)
* RSA(Rivest-Shamir-Adleman)

###DES

* 对称加密-共用一个密钥来加密和解密数据
* 分组加密-算法每次都处理固定长度的数据段，称为分组。DES分组大小为64位，不足64位则填充

混乱与扩散

* 混乱的目的是为了隐藏任何明文同密文或者密钥之间的关系
* 扩散的目的是使明文中的有效位和密钥一起组成尽可能多的密文
